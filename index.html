wagablog

Wagablog is my personal blog that I write posts on in Japanese. The site has basic CRUD functionalitu. It is built with ruby on rails using a PostgreSQL database. I styled it with tailwind and have it hosted on a Linux server on Microsoft Azure.

Github

mimizu

Mimizu is a site that allows users to search through and sort books from the public domain online Japanese library Aozora Bunko. The sight fetches data from a RESTful API that I wrote with Go. The data is stored in a PostgreSQL database. I have it hosted on a Linux server on AWS.

Github

text n driver

Text n driver is a web app game that is supposed to test the users ability to multitask. It was completed for a software engineering class in university. Along with the website, I wrote documents on the phases of requirements, design and implementation. The website was written using the LAMP stack.
